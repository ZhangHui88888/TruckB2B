# 图片资源说明

## 必需的图片文件

### 1. OG Image（社交分享图）
- **文件名**: `og-image.jpg`
- **尺寸**: 1200 x 630 像素
- **用途**: 网站链接在社交媒体（WhatsApp、Facebook、LinkedIn）分享时显示的预览图
- **建议内容**:
  - 公司 Logo
  - 主要产品图片（卡车大灯、后视镜等）
  - 品牌色背景
  - 简短标语："Heavy Truck Parts Supplier"

### 2. Apple Touch Icon
- **文件名**: `apple-touch-icon.png`
- **尺寸**: 180 x 180 像素
- **用途**: iPhone/iPad 添加到主屏幕时显示的图标

### 3. Logo
- **文件名**: `logo.png`
- **尺寸**: 200 x 60 像素（建议）
- **用途**: 网站 header 和 Schema.org 数据

## 图片优化建议

1. **压缩**: 使用 TinyPNG 或 Squoosh 压缩图片
2. **格式**: 
   - 照片用 `.jpg`（质量 80-85%）
   - Logo/图标用 `.svg` 或 `.png`
3. **命名**: 使用小写字母和连字符，如 `volvo-headlamp.jpg`
